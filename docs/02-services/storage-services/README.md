## Overview

Azure storage services provide the foundation for the Cloud Scale Analytics platform, enabling scalable, secure, and cost-effective data storage across multiple tiers and access patterns.

## Storage Services

| Service | Description | Documentation |
|---------|-------------|---------------|
| 🏞️ **Azure Data Lake Storage Gen2** | Hierarchical namespace storage optimized for analytics workloads | [View](./azure-data-lake-gen2/README.md) |
| 🌐 **Azure Cosmos DB** | Globally distributed, multi-model database service | [View](./azure-cosmos-db/README.md) |
| 🗄️ **Azure SQL Database** | Managed relational database service | [View](./azure-sql-database/README.md) |

## Key Capabilities

- **Scalable Storage**: Petabyte-scale storage with cost-effective tiering
- **Security**: Fine-grained access control with Azure AD integration
- **Performance**: High-throughput and low-latency for analytics workloads
- **Integration**: Native integration with Azure Synapse Analytics, Azure Databricks, and other Azure services
