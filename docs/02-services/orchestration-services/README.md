# 🔧 Orchestration Services

> **🏠 [Home](../../../README.md)** | **📖 [Services](../README.md)** | **🔧 Orchestration Services**

## Overview

Azure orchestration services enable the automation, scheduling, and coordination of data workflows and business processes within the Cloud Scale Analytics platform.

## Orchestration Services

| Service | Description | Documentation |
|---------|-------------|---------------|
| 🔄 **Azure Data Factory** | Cloud-based ETL and data integration service | [View](./azure-data-factory/README.md) |
| ⚙️ **Azure Logic Apps** | Low-code workflow automation and integration | [View](./azure-logic-apps/README.md) |

## Key Capabilities

- **Data Pipeline Orchestration**: Design and schedule complex data transformation workflows
- **Integration**: Connect to hundreds of data sources and services
- **Monitoring**: Built-in monitoring, alerting, and retry capabilities
- **Scalability**: Auto-scale compute resources to handle varying workload demands
