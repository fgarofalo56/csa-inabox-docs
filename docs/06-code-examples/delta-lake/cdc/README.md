# Change Data Capture (CDC)

This section provides code examples for implementing Change Data Capture patterns with Delta Lake.

## Overview

Change Data Capture enables tracking and processing of data changes for incremental data pipelines.

## Contents

- [Change Data Capture](change-data-capture.md) - CDC implementation patterns and examples

## Key Concepts

### CDC Patterns

- Merge operations for upserts
- Delete detection and handling
- Change tracking queries
- Time travel for auditing

### Use Cases

- Data warehouse incremental loads
- Real-time data synchronization
- Audit trail maintenance
- Data replication
