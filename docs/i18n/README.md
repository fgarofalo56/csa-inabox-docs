# Internationalization (i18n)

This section contains resources for documentation internationalization and localization.

## Overview

Guidelines and resources for translating documentation into multiple languages.

## Contents

- [Translation Status](TRANSLATION_STATUS.md) - Current translation progress
- [Installation](installation.md) - Setting up translation tools

## Supported Languages

The documentation is prepared for translation into:

- English (primary)
- French
- German
- Spanish
- Portuguese (Brazil)
- Japanese
- Chinese (Simplified)

## Translation Guidelines

### Getting Started

1. Review the translation status document
2. Set up the translation environment
3. Follow the style guide for your language
4. Submit translations for review

### Quality Standards

- Technical accuracy
- Consistent terminology
- Cultural adaptation
- Style consistency
