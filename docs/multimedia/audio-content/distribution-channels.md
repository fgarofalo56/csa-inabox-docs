## ðŸ“Š Distribution Workflow

### 1. Pre-Publication (1-2 hours)

- [ ] Final audio file exported (MP3, 320 kbps)
- [ ] Transcript completed
- [ ] Show notes written
- [ ] Episode artwork created (3000x3000px)
- [ ] Metadata prepared
- [ ] SEO optimization completed

### 2. Upload to Hosting (30 min)

- Upload audio file
- Add episode title and description
- Set publication date/time
- Add episode artwork
- Insert chapter markers (if supported)
- Add links and resources

### 3. RSS Feed Update (Automatic)

- Hosting service updates RSS feed
- Platforms pull new episode automatically
- Typically available within 24 hours

### 4. Manual Distribution (1 hour)

- Upload to YouTube with static image
- Post to corporate channels
- Share on social media
- Send email newsletter
- Update website

### 5. Promotion (Ongoing)

- Social media posts (Twitter, LinkedIn)
- Blog post with episode summary
- Email to subscriber list
- Internal company communication
- Guest cross-promotion
