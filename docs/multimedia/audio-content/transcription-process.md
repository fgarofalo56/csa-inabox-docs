## üîÑ Recommended Workflow

### Step 1: Automated Draft (2-24 hours)

1. Upload audio to transcription service
2. Specify:
   - Speaker count
   - Technical vocabulary
   - Timestamps required
3. Receive initial transcript (90-95% accurate)

### Step 2: First Pass Edit (1-3 hours per hour of audio)

- Correct obvious errors
- Add speaker labels
- Format timestamps consistently
- Note [MUSIC] and [SOUND EFFECTS]
- Fix technical terminology
- Add paragraph breaks

### Step 3: Technical Review (30-60 minutes)

Subject matter expert reviews for:
- Technical accuracy
- Correct terminology
- Acronym expansions
- Resource references
- Numerical accuracy

### Step 4: Copy Edit (20-30 minutes)

Final proofread for:
- Grammar and punctuation
- Consistency
- Formatting
- Accessibility compliance

### Step 5: Format Export (10-15 minutes)

Create multiple formats:
- .txt (plain text)
- .docx (rich text)
- .pdf (formatted)
- .srt (captions, if video)

### Step 6: Publication (10 minutes)

- Upload to content management system
- Link from audio player
- Add to transcript library
- Make downloadable
