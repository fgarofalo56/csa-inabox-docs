## üìê Layout Standards

### Slide Templates

**Title Slide**:
```text
- Azure logo (top left)
- Presentation title: 54pt Bold
- Subtitle: 32pt Regular
- Presenter info: 24pt Light
- Date: 20pt Light
- Background: White or subtle gradient
```

**Content Slides**:
```text
- Slide number (bottom right)
- Title: 44pt Bold (left-aligned)
- Body: 24pt Regular
- Maximum 6 bullets
- 50% white space minimum
- Consistent margins (1 inch all sides)
```

**Section Divider**:
```text
- Large section title: 60pt Bold
- Azure Blue background
- White text
- Minimal additional content
```

### Grid System

**Master Layout Grid**:
```text
Slide dimensions: 16:9 (10" √ó 5.625")

Margins:
- Top: 0.75"
- Bottom: 0.5"
- Left/Right: 0.5"

Columns: 12-column grid
Gutter: 0.2"
```

### Visual Hierarchy

**Hierarchy Principles**:
1. **Title**: Largest, bold, top of slide
2. **Subheadings**: Medium, semibold
3. **Body text**: Smaller, regular weight
4. **Captions**: Smallest, light weight

**Spacing Rules**:
- Title to content: 0.4"
- Between sections: 0.3"
- Bullet spacing: 1.5√ó line height
- Image padding: 0.2" minimum

## ‚ú® Motion and Animation

### Transition Guidelines

**Approved Transitions**:
- Fade (most common)
- Push (for section changes)
- Morph (PowerPoint 365 only)
- None (for accessibility)

**Avoid**:
- Flashy transitions (Cube, Flip, etc.)
- Different transitions per slide
- Sound effects
- Automatic timing

### Animation Best Practices

**When to Animate**:
- Reveal bullet points progressively
- Highlight key data points
- Show process flows step-by-step
- Bring attention to specific elements

**Animation Settings**:
- Duration: 0.5-0.75 seconds
- Effect: Fade, Wipe, or Appear
- Trigger: On Click (not automatic)
- Direction: Consistent throughout
