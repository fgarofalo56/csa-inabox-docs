### Attribution Templates

When attribution is required, use these formats:

**Photo Attribution:**
```html
Photo by <a href="[photographer_url]">[Photographer Name]</a>
on <a href="[source_url]">[Source Name]</a>
```

**Video Attribution:**
```html
Video by <a href="[creator_url]">[Creator Name]</a>
from <a href="[source_url]">[Source Name]</a>
```

**Audio Attribution:**
```html
Music: "[Track Name]" by [Artist Name]
Source: [Source Name]
License: [License Type]
```

## üìä Asset Usage Report

### Generate Usage Report

```python
# generate_stock_report.py
from pathlib import Path
import yaml

def generate_stock_usage_report():
    """Generate report of stock asset usage"""
    assets = []

    # Scan all asset metadata files
    for metadata_file in Path('./docs/multimedia').rglob('*-metadata.yaml'):
        with open(metadata_file) as f:
            asset = yaml.safe_load(f)
            assets.append(asset)

    # Group by source
    sources = {}
    for asset in assets:
        source = asset.get('source', 'Unknown')
        if source not in sources:
            sources[source] = []
        sources[source].append(asset)

    # Generate report
    report = "# Stock Asset Usage Report\n\n"
    for source, items in sources.items():
        report += f"## {source}\n"
        report += f"Total assets: {len(items)}\n\n"
        for item in items:
            report += f"- {item['filename']}"
            if item.get('attribution_required'):
                report += " ‚ö†Ô∏è Requires attribution"
            report += "\n"
        report += "\n"

    return report

if __name__ == "__main__":
    print(generate_stock_usage_report())
```
