## Overview

Comprehensive checklist for reviewing video tutorials before publication, ensuring quality, accuracy, and consistency.

## Pre-Production Review

### Script Review
- [ ] Learning objectives clearly defined
- [ ] Content accurate and up-to-date
- [ ] Azure-specific examples included
- [ ] Duration target realistic
- [ ] Prerequisites clearly stated
- [ ] Related resources linked

### Storyboard Review
- [ ] Visual elements support narrative
- [ ] Transitions planned appropriately
- [ ] Animation timing specified
- [ ] Brand guidelines followed
- [ ] Accessibility considered
- [ ] Technical diagrams accurate

## Production Review

### Video Quality
- [ ] Resolution: 1080p minimum (4K preferred)
- [ ] Frame rate: 30fps or 60fps consistent
- [ ] Bitrate: Adequate for platform
- [ ] Color grading consistent
- [ ] No visual artifacts or glitches
- [ ] Watermarks/branding applied correctly

### Audio Quality
- [ ] Narration clear and audible
- [ ] Background music appropriate volume
- [ ] No audio clipping or distortion
- [ ] Consistent audio levels throughout
- [ ] No background noise
- [ ] Sound effects synchronized

### Screen Recordings
- [ ] UI elements clearly visible
- [ ] Mouse cursor visible when needed
- [ ] No personal information visible
- [ ] Browser zoom at appropriate level
- [ ] Annotations and callouts clear
- [ ] Code snippets readable

## Content Review

### Technical Accuracy
- [ ] Code examples tested and working
- [ ] Azure portal UI current
- [ ] Commands verified in latest versions
- [ ] Best practices current
- [ ] Security recommendations valid
- [ ] Cost information accurate

### Instructional Quality
- [ ] Clear step-by-step progression
- [ ] Complex concepts explained simply
- [ ] Real-world examples included
- [ ] Common pitfalls addressed
- [ ] Troubleshooting tips provided
- [ ] Next steps clearly stated

### Accessibility
- [ ] Captions accurate (99%+ accuracy)
- [ ] Audio descriptions for visual content
- [ ] Transcript available
- [ ] High contrast visuals
- [ ] No rapid flashing (seizure risk)
- [ ] Font sizes readable on mobile

## Post-Production Review

### Editing Quality
- [ ] Smooth transitions between scenes
- [ ] Chapter markers added
- [ ] Lower-thirds consistent
- [ ] On-screen text legible
- [ ] Pacing appropriate
- [ ] No dead air or awkward pauses

### Branding & Compliance
- [ ] Azure branding guidelines followed
- [ ] Company logo placement correct
- [ ] Copyright notices included
- [ ] Music licensed appropriately
- [ ] Third-party trademark usage approved
- [ ] Disclaimers added if needed

### Metadata & Publishing
- [ ] Title optimized for search
- [ ] Description comprehensive
- [ ] Tags relevant and complete
- [ ] Thumbnail compelling and on-brand
- [ ] Category appropriate
- [ ] Playlist assignment correct

## Final Checklist

### Quality Assurance
- [ ] Watched entire video start to finish
- [ ] Tested on multiple devices (desktop, tablet, mobile)
- [ ] Verified in multiple browsers
- [ ] Checked at different quality settings
- [ ] Confirmed links in description work
- [ ] Verified captions synchronized

### Documentation
- [ ] Production notes updated
- [ ] Asset inventory complete
- [ ] Source files archived
- [ ] Version history documented
- [ ] Feedback template prepared
- [ ] Analytics tracking configured

### Approval
- [ ] Technical reviewer approved
- [ ] Content owner approved
- [ ] Legal/compliance review (if required)
- [ ] Stakeholder sign-off obtained
- [ ] Publication date confirmed
- [ ] Promotion plan ready
